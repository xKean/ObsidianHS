### Definition
Diese Systeme bestehen aus einer Kombination mehrerer Softwaresysteme, die zusammenarbeiten, um eine gemeinsame Aufgabe zu erfüllen. Das System von Systemen führt mehrere Systeme zusammen und ermöglicht die Zusammenarbeit.


### Beispiel
Ein Smart-Home-System besteht aus verschiedenen Komponenten wie Beleuchtungssteuerung, Heizungs- und Kühlungsregelung.
Jedes dieser Systeme ist eine eigenständige Software / Hardware und kann durch das System von Systemen zu einem großen Smart-Home-System zusammengefasst werden.
